## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

React 17.0.2 or higher

### Setup

1. git clone https://github.com/mnekx/react-to-do.git
2. cd to-do
3. npm install
4. npm start to serve into local 8080 port

### Install

1. git clone https://github.com/mnekx/react-to-do.git
2. cd to-do
3. npm install
4. npm start to serve into local 8080 port

### Run tests
run: npm run test
